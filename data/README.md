# Offline dictionaries

Dictionaries are downloaded from open sources and stored locally:
- `ru-open.txt` — Russian wordlist (source: hunspell-ru package data)
- `en-open.txt` — English wordlist (source: [sujithps/Dictionary](https://github.com/sujithps/Dictionary), file: `Oxford English Dictionary.txt`)

Both files are used fully offline in the browser.

`en-open.txt` now stores tab-separated pairs: `word<TAB>definition`.

> Note: these are wordlist dictionaries. They provide broad vocabulary coverage for Scrabble/word-game matching.

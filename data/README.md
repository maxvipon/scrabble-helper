# Offline dictionaries

Dictionaries are downloaded from open package sources and stored locally:
- `ru-open.txt` — Russian wordlist (source: hunspell-ru package data)
- `en-open.txt` — English wordlist (source: hunspell-en-us package data)

Both files are used fully offline in the browser.

> Note: these are wordlist dictionaries. They provide broad vocabulary coverage for Scrabble/word-game matching.

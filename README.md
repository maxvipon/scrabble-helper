# Scrabble Helper (RU/EN)

Небольшое веб-приложение для подбора слов в формате «Скрэббл / Эрудит».

## Онлайн-версия

- https://maxvipon.github.io/scrabble-helper/

## Возможности

- Русский и английский словари.
- Поиск слов по набору букв.
- Маска поиска: `_` — одна любая буква, `*` — любое количество букв.
- Сортировка по очкам.
- Фильтр по длине слова.

## Локальный запуск

> Нужен статический HTTP-сервер: словари загружаются через `fetch`, поэтому `index.html` нельзя открывать напрямую через `file://`.

```bash
python3 -m http.server 8080
```

Откройте: http://localhost:8080
